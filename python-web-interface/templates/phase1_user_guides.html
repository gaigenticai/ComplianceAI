Customer Onboarding Guide

This guide documents Phase 1: intake and customer onboarding workflows.

1. Purpose

Collect required identity and proof-of-address documentation, validate formats and extract metadata.

2. Supported Document Types

  • Passport, National ID, Driver's License (PDF, JPG, PNG)
  • Utility bill, Bank statement (proof of address)
  • Corporate registrations and incorporation docs (for businesses)

3. Processing Steps

  1. Secure upload (multipart/form-data to /api/documents/upload)
  2. Automatic OCR & text extraction
  3. Metadata validation (dates, MRZ parsing where applicable)
  4. Initial document classification and case creation

4. Troubleshooting

If OCR fails, ensure image DPI & lighting are sufficient. For PDF issues, confirm embedded fonts are not encrypted.
