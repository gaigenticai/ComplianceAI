System Architecture
Architecture Overview

The ComplianceAI system follows a microservices architecture with clear separation of concerns:

  • Regulatory Intelligence Agent: Handles data ingestion and regulatory updates
  • Intelligence & Compliance Agent: Processes rules and compliance logic
  • Decision & Orchestration Agent: Manages report generation and delivery
  • Intake & Processing Agent: Handles document processing and data intake
  • Web Interface: Provides user interaction and monitoring
