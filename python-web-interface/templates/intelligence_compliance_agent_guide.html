Overview

The Intelligence & Compliance Agent compiles regulatory obligations into executable rules, resolves jurisdictional conflicts, and publishes rules to downstream compliance engines.

Key Components

  • Rule Compiler — Converts obligations into JSON-Logic policies.
  • Kafka Consumer — Subscribes to regulatory updates and triggers recompilation.
  • Jurisdiction Handler — Filters and orders rules by jurisdiction precedence.
  • Overlap Resolver — Detects and resolves overlapping obligations.
  • Audit Logger — Records rule provenance and changes.

API Endpoints

GET /health
GET /api/intel/rules
POST /api/intel/recompile
GET /metrics

Usage

Use the /api/intel/recompile endpoint to trigger rule compilation after significant regulatory updates. Rules are published to Kafka topic regulatory.rules.
