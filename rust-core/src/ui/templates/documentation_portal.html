SDK & Libraries

Python SDK
from complianceai import KYCClient

client = KYCClient(api_key="your_key")
result = client.process_kyc(
    customer_data=customer_data,
    callback_url="https://your-app.com/webhook"
)
Node.js SDK
const ComplianceAI = require('complianceai');

const client = new ComplianceAI('your_key');
const result = await client.processKYC({
  customerData: customerData,
  callbackUrl: 'https://your-app.com/webhook'
});
