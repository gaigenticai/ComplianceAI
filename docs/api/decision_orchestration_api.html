üìäReport Generation API

POST /api/v1/reports/generate
Generate a regulatory compliance report with specified parameters and format options.

Request Body

Field Type Required Description
report_type string required Report type: FINREP, COREP, DORA, AML_SAR, FATCA, CRS
institution_id string required Unique identifier of the reporting institution
reporting_date date required Reporting period end date (YYYY-MM-DD)
format string optional Output format: XBRL, CSV, JSON, PDF (default: XBRL)
jurisdiction string optional Reporting jurisdiction (auto-detected if not provided)
template_version string optional Specific template version to use
validation_level string optional Validation level: basic, standard, strict (default: standard)
auto_submit boolean optional Automatically submit after generation (default: false)
