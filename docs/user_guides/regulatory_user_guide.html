    Troubleshooting

    Common Issues and Solutions

    Report Generation Issues

    Problem: Report generation fails with data validation errors
    Solution: Check data source connectivity and data completeness
  • Verify data source connections are active
  • Check data completeness using the data quality dashboard
  • Review validation error messages in detail
  • Contact system administrator for data source issues
  • Use manual data entry for missing critical fields

    • Delivery Failures

    Problem: SFTP delivery fails with authentication errors
    Solution: Verify SFTP credentials and network connectivity
  • Check SFTP server connectivity using test connection
  • Verify username, password, and key file are correct
  • Confirm firewall rules allow SFTP traffic (port 22)
  • Test file upload permissions on target directory
  • Review SFTP server logs for detailed error information

    • Jurisdiction Configuration Issues

    Problem: New jurisdiction not appearing in dropdown
    Solution: Refresh jurisdiction cache and verify permissions
  • Clear browser cache and refresh the page
  • Check user permissions for jurisdiction management
  • Verify jurisdiction has been activated in system configuration
  • Contact administrator to confirm jurisdiction setup

    • Error Codes and Meanings

    Error Code Description Action Required
    RPT-001 Data source unavailable Check data source connectivity
    RPT-002 Validation rule failure Review data quality issues
    DLV-001 SFTP authentication failed Verify SFTP credentials
    DLV-002 EBA API rate limit exceeded Wait and retry submission
    JUR-001 Jurisdiction configuration error Contact system administrator
